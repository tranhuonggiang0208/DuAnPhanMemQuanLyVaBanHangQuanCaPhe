# Phần Mềm Quản Lý & Bán Hàng Quán Cà Phê
## Giới thiệu
Đồ án môn học xây dựng hệ thống quản lý và bán hàng cho quán cà phê bằng Java Swing, sử dụng SQL Server để lưu trữ dữ liệu.
## Công nghệ sử dụng
* Java (JDK 8/11/17)
* Java Swing
* Microsoft SQL Server
* JDBC
* Apache NetBeans 25
## Chức năng chính
* Đăng nhập và phân quyền (Admin / Staff)
* Quản lý bán hàng (gọi món, thanh toán)
* Quản lý hóa đơn
* Quản lý sản phẩm
* Quản lý nhân viên
* Quản lý tài khoản
* Quản lý ưu đãi
* Thống kê doanh thu
## Hướng dẫn chạy dự án
1. Clone project:
   git clone https://github.com/tranhuonggiang0208/DuAnPhanMemQuanLyVaBanHangQuanCaPhe.git
3. Tạo database `QuanLyQuanCaPhe` trong SQL Server
4. Chạy file `script.sql`
5. Cấu hình lại thông tin kết nối trong `DatabaseConnection.java`
6. Mở project bằng NetBeans 25 và nhấn Run (F6)
